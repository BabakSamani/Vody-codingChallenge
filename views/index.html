Welcome to Media API

Instructions:

The following is the instruction for using this API and helps you how to search for a media, a movie or a show in this API.

GET:  /api/movie/${id}

Description:

Returns a JSON data of a movie type media based on the movie id and it contains queryable parameters.

Request:

http://142.93.195.130/api/${user_token}/movie/5bbfd7fd60e7e379d49010e6

Respond:

{ "_id": "5bbfefea60e7e31fed1f386d", "duration": "22 min", "episodes": "30", "genre": "Animation, Comedy", "media type": "show", "release year": "1989-2013", "synopsis": "The satiric adventures of a working-class family in the misfit city of Springfield.", "title": "The Simpsons" }

GET:  /api/${user_token}/search/query=${title}

Description:

Returns a JSON data of a media, show or movie, based on the title of the media and it contains queryable parameters.

Request:

http://142.93.195.130/api/${user_token}/search/query=1 Night

Respond:

{ "_id": "5bbfd7fe60e7e379d49010f4", "duration": "80 min", "genre": "Drama, Romance", "media type": "movie", "release year": "2016", "synopsis": "Past and present collide when two couples explore love over the course of one night at a hotel.", "title": "1 Night" }
